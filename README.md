# Pathfinding con Algoritmo A* (C++)

Questo progetto contiene un'implementazione in C++ dell'algoritmo di ricerca A* (A-star). Il programma simula un esploratore che trova il percorso più breve all'interno di un labirinto (rappresentato tramite mappa testuale) calcolando l'euristica e tenendo traccia del numero di passi effettuati.

## Compilazione ed esecuzione
```bash
g++ main.cpp -o intelligenza
./intelligenza
